## Details


* **OS:** Bedrock Linux {Void + Arch}
 
* **WM:** bspwm, i3-gaps
 
* **Terminal:** [st](https://gitlab.com/Tanish2002/st-tanish2002)

* **Shell:** zsh
 
* **Bar:** polybar
 
* **Compositor:** Picom

## Installation

### **Clone my repo**
```
$ cd
$ git clone https://gitlab.com/Tanish2002/dot-files
$ cd dotfiles
``` 

### **Use stow**
```
$ stow <WM-Name> -t ~/
$ stow Wallpapers -t ~/
$ stow bin -t ~/

Do basic stuff like using pywal or xrdb and adding bin to $PATH
```
