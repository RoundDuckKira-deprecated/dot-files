# Installation

**Clone my repo**
```
$ cd
$ git clone https://gitlab.com/Tanish2002/dot-files
$ cd dotfiles
``` 

**Use stow**
```
$ stow Bspwm
$ stow Wallpapers
```
**And Voilla That's All!!**`
