# Details


* **OS:** Manjaro Linux
 
* **WM:** bspwm, i3-gaps
 
* **Terminal:** Kitty
 
* **Terminal Font:** Fira Code Mono
 
* **Shell:** zsh
 
* **Bar:** polybar
 
* **Compositor:** sdhand-compton


# Installation

**Clone my repo**
```
$ cd
$ git clone https://gitlab.com/Tanish2002/dot-files
$ cd dotfiles
``` 

**Use stow**
```
$ stow <WM-Name>
$ stow Wallpapers
$ stow bin
```
**And Voilla That's All!!**`
